# projeto-agrinho 2026 🌱
Labelly Felde K Dos Santos 🌱 Agrinho 2026 – Relação entre o Campo e a Cidade

# 🎯 Objetivo do Projeto
Este projeto foi desenvolvido para o Agrinho 2026, com o tema voltado à relação entre o campo e a cidade. O objetivo é apresentar, de forma interativa e informativa, como esses dois ambientes estão conectados através da tecnologia, da produção agrícola, do comércio e da cooperação entre as pessoas.

# 🖥️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Responsividade com Media Queries
* Animações e interatividade com CSS e JavaScript

# 📸 Créditos das Imagens
* Todas as imagens utilizadas no projeto foram criadas com auxílio de Inteligência Artificial (IA), contribuindo para representar visualmente o tema da conexão entre o campo e a cidade.
